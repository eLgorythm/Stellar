## SELF-PAIRING CONTEXT
Aplikasi ini dirancang untuk melakukan "self-pairing", di mana aplikasi yang berjalan di perangkat Android melakukan proses pairing dengan layanan Wireless Debugging pada perangkat fisik yang sama, bukan ke perangkat lain. Aplikasi sebagai client dan adb sebagai server.

Aplikasi sebagai Client melakukan pairing ke adbd server (Wireles Debuging) di Android device yang sama melalui localhost.

Untuk android 11+ terutama 13+
